## 📌 Problem

Every year, thousands of startup and project ideas fail because teams start building without validating whether the idea is unique, feasible, or already existing.

Current idea validation involves:
- Manual research  
- Market analysis  
- Patent checking  

This process is slow, inefficient, and often unreliable.

---

## 💡 Solution

**InnoCheck Enterprise** is a GenAI-powered multi-agent system that analyzes, evaluates, and improves innovation ideas in seconds.

It simulates multiple intelligent agents to provide:
- Idea understanding  
- Market research insights  
- Patent similarity analysis  
- Novelty scoring  
- Final decision (GO / MODIFY / REJECT)  

---

## ⚙️ How It Works

1. User enters an idea  
2. AI analyzes market and similar concepts  
3. System generates novelty score and insights  
4. Final decision is provided:
   - ✅ GO  
   - ⚠️ MODIFY  
   - ❌ REJECT
